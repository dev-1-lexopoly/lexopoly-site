3. Three-Tier Confidence System

THE KILLER FEATURE: Show EPA transparency about record quality. Based on EPA's own Audit Policy.

  • DOCUMENTED: Based on invoices, receipts, manifests (best evidence)
  • ESTIMATED: Calculated from project records (reasonable approximation)
  • RECONSTRUCTED: Best guess from memory (honest recollection)
  • Color-coded badges in interface
  • Automatic confidence indicators in exports
  • Transparent timestamping (EPA sees when records were created)
  • Legal framework: EPA Audit Policy supports good faith reconstruction
